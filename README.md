# Skyline workspace

Bun monorepo for the Skyline TypeScript SDK. Published packages live under `packages/`.

```bash
bun install
bun run typecheck
bun run build
```

Install the batteries-included SDK: `bun add skyline-ts`

Or pick platforms à la carte: `@skyline-ts/core`, `@skyline-ts/imessage`, `@skyline-ts/slack`, etc.

See `packages/skyline-ts/README.md` for full documentation.
